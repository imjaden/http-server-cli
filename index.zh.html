http-server-cli

忘掉端口，只管预览。

🔄 自动分配端口· 🧠 智能首页· 📋 项目追踪· ⚡ 零依赖

pip install http-server-cli

零依赖的 本地 HTTP 预览 CLI 工具，自动记住你启动过的服务。 只需 hs . -o —— 不用再记端口号、不用纠结“哪个项目占用了 8080”、不用翻进程管理器去 kill。

从 8080 开始分配，端口冲突时自动递增。持久化追踪每次启动的服务。 一键查看所有运行中的服务、按端口或路径关闭、打开 Web 管理面板 —— 一个工具搞定。

▶ 启动
$ hs . -o
✅ http://jaden.local:8081 → ~/project-alpha
$ hs ./dist -i app.html -o
✅ http://jaden.local:8082/app.html → ~/project/dist
$ hs . --index 'snapshots/*.html' -o
✅ http://jaden.local:8083/snapshots/latest.html
▶ 查看
$ hs list
📊 共 2 个 HTTP 服务
http://jaden.local:8080 → ~/project-beta
http://jaden.local:8081 → ~/project-alpha
$ hs list --json
[{"port":8080,"path":"~/beta",...}]
$ hs status
✅ 守护进程运行中 (PID: 12345)
$ hs dashboard -o
📊 管理面板 → http://jaden.local:8180
▶ 关闭
$ hs kill 8080
🛑 已终止 PID: 45678
✅ 端口 8080 已关闭
$ hs kill ~/project-alpha
✅ ~/project-alpha 已关闭
$ hs kill-all
✅ 已关闭 3 个服务
