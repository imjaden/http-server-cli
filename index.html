http-server-cli

Forget ports, just preview.

🔄 Auto port· 🧠 Smart index· 📋 Project tracking· ⚡ Zero dep

pip install http-server-cli

A zero-dependency CLI for local HTTP preview that remembers what you're serving. Run hs . -o — no port juggling, no "which project was on 8080?", no "how do I kill it?".

Starts at 8080, auto-increments on conflict. Tracks every server you launch. List all running services, kill by port or path, or open the web dashboard — all in one tool.

▶ Start
$ hs . -o
✅ http://jaden.local:8081 → ~/project-alpha
$ hs ./dist -i app.html -o
✅ http://jaden.local:8082/app.html → ~/project/dist
$ hs . --index 'snapshots/*.html' -o
✅ http://jaden.local:8083/snapshots/latest.html
▶ View
$ hs list
📊 Total 2 HTTP services
http://jaden.local:8080 → ~/project-beta
http://jaden.local:8081 → ~/project-alpha
$ hs list --json
[{"port":8080,"path":"~/beta",...}]
$ hs status
✅ Daemon running (PID: 12345)
$ hs dashboard -o
📊 Dashboard → http://jaden.local:8180
▶ Kill
$ hs kill 8080
🛑 Terminated PID: 45678
✅ Port 8080 closed
$ hs kill ~/project-alpha
✅ ~/project-alpha closed
$ hs kill-all
✅ 3 service(s) closed
